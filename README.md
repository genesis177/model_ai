# AI Model Building with DL4J

Этот проект представляет собой Java-приложение на Spring Boot, использующее DL4J (Deeplearning4j) для обучения нейронной сети, которая предсказывает,подходит ли игрок для команды.

Приложение поддерживает **CRUD-операции** для данных о производительности игроков, хранящихся в базе данных **MySQL**, и автоматически переобучает AI-модель при изменении данных.

---

##  Возможности

- **CRUD-операции**  
  Добавление, обновление, удаление и получение данных о производительности игроков  

- **AI-модель**  
  Используется нейронная сеть (DL4J) для предсказания пригодности игрока  

- **Автоматическое переобучение модели**  
  Модель переобучается при каждом изменении данных  

- **Интеграция с MySQL**  
  Хранение данных игроков в реляционной базе данных  

- **REST API**  
  Предоставляет эндпоинты для управления данными игроков  

---

## Структура проекта

```bash
src/main/java/com/ictss
├── ai              # логика модели DL4J
├── controller      # REST-контроллеры
├── dto             # объекты передачи данных (DTO)
├── model           # JPA-сущности
├── repository      # репозитории Spring Data JPA
└── service         # слой бизнес-логики
```

## Используемые технологии

- Java 17  
- Spring Boot  
- Deeplearning4j (DL4J)  
- MySQL  
- Spring Data JPA  

---

## Описание

Проект демонстрирует, как можно интегрировать машинное обучение в backend-приложение на Java.  
Модель автоматически адаптируется к новым данным, что делает систему самообучающейся.
